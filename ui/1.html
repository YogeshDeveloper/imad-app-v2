Introduction

Path Tracker is a cloud and mobile based platform. It helps organizations Plan, Analyse and Track its Field Operations (Sales and Service) by enabling Real Time Tracking of its Field Workforce.

Our application helps you create new orders, view all existing orders in a List/Map based view, group and assign them to your agents for processing.

Our mobile application helps guide the field agents with the location of customers with inbuilt map that helps him navigate directly to the desired location. It helps the manager to track and communicate with agent in real time through Mobile App instantaneously

Our mobile application allows customer to track the location of the filed agent and communicate with agent.

We also support integrating our software with other 3rd party software

Sign Up

Add New Agent

Agent Management section contains all the details about agent. You can add new agent to the account, view them, monitor their day to day activities with the help of a map.

Add New Agent




Click the Agent Management on side navigation menu bar

Click Add new agent.

Fill in the following fields

Name (Name of the agent)

Email Address (Email Id of the agent) (Optional)

Phone number (Mobile Number of the agent) and

Click Submit



An agent will be created and a notification will appear on the top of the page which reads Agent created successfully for “Agent Name” Details

Clicking Details will navigate you to the agent details page

You can view see more details about Agent details page on Agent Quickview Section.

View All Agents

‘View all agents’ lists all the registered agents associated to your account and shows their current status



Click Agent Management on the side navigation menu bar

Click View all agents

Herein, you will see the agent list which contains the details of all the registered agents that are associated to your account. It gives the information such as Current Location, Current Assignments, Duty Status etc.,


Click Actions on Agents list on the the right top of the page, then Click Refresh

(It Refreshes the page to get updated information)

You can see the “Last refresh time” below the Agent List

Agent Name


It lists the details of all the registered agents that are associated to your account.

If you click any "Agent Name" on the Agent Name Column the page will navigate to agent details page.

You can view see more details about Agent details page on Agent Quickview Section.

Location Active


It gives the status update on the agent location.

Yes- This status is shown if we have had an update on the agent’s location within the last 30 minutes.

No- This status is shown if we haven’t had an update on the agent’s location within the last 30 minutes

NA-This status is shown if we haven’t had an update on the agent’s location for the last 24 hours or more

Last Update


This tab displays the date and time as to when the agent details were last updated. If it reads “NA” it implies that the agent’s status details have not been available for more than 30 days.

Data Sync


This tab displays the most recent time when the Data was synchronized. Example: Wed 21 Dec 3:24:12 pm

More details to be updated

Location


This tab displays the most recent location of the agent.
If it reads “NA” it implies that the agent has not been active for more than 24 hours.

Status


This tab displays the status of the current assignment that the Agent is working on.

On-duty/Started


- When the agent is on duty and has started for the current assignment.

On-duty/Reached


-When the agent has reached the place of the current assignment.

Idle


- When the agent has finished the assignment or not yet started for the new assignment.

Current Assignments


This tab displays the number of orders that have been assigned to a particular agent



Duty Status


This tab displays the duty status of an agent. On would imply that the agent currently on duty and Off would imply that the agent currently off duty

This can also be done manually by the agent on their mobile app

An agent can toggle on or off their duty status as shown





Map

Balloon Icon
